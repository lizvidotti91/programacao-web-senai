# Sistema de Post-its

* Você pode ver este projeto no meu [CodePen](https://codepen.io/lizvidotti91/pen/yLOwXRw); 

## Objetivo

Criação de Página HTML com JavaScript dinamico.

## Tecnologias usadas

* JavaScript; 
* HTML; 

## Descrição

* Criar um sisteminha de todos com postits amarelos; 
* O usuário vai digitar algo no input e ao clicar no botão; 
* Deverá ser criado um postit com aquela tarefa; 
* Não esqueça de limpar o input ao final da criação do postit.
