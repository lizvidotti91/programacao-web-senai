Regras

Um jogo infantil, muito popular, é o 21 de boca. O jogo é jogado da seguinte forma: o primeiro jogador diz um número N, que pode ser 1, 2 ou 3. O segundo jogador pode então dizer um número N1 tal que seja N + 1; N + 2; N + 3. E assim por diante, os jogadores se alternam, dizendo sempre um número entre os três números consecutivos ao número anterior. O jogador que disser 21 ganha o jogo. Caso a soma seja maior do que 21, o adversário ganha o jogo.
